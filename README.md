# Griddo Atlas Marketplace

Claude Code plugins for university data intelligence.

## Installation

```bash
/plugin marketplace add griddo/atlas-marketplace
```

## Available Plugins

### university-atlas

Search, enrich, and analyze university data via the Griddo Atlas MCP server.

```bash
/plugin install university-atlas@griddo-atlas
```

See [plugins/university-atlas/README.md](plugins/university-atlas/README.md) for details.
